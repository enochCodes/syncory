## Installation

1. Clone the repository:

    ```sh
    git clone https://github.com/enochCodes/syncory.git
    ```

2. Navigate to the project directory:

    ```sh
    cd syncory
    ```

3. Install dependencies:

    ```sh
    npm install
    ```
